# Sacramento Dental Medicine Redesign

A clean, modern Next.js redesign for Sacramento Dental Medicine in Antelope, CA.

## Audit Summary

The source site has useful content but weak hierarchy. The live homepage currently leads with a large hero, tiny navigation, low-contrast calls to action, and long blank patterned sections before important patient information appears. Mobile contains more useful text, but the page still stretches too long before core actions.

The redesign focuses on:

- Clear first-screen identity: clinic name, address, call, and booking.
- Real proof: team photo, doctor portraits, patient reviews, technology, and hours.
- Patient-first grouping: preventive, cosmetic, restorative, and oral surgery paths.
- Faster decisions: repeated book, call, and directions actions.
- Modern visual system: warm white, ink, teal, sage, coral, and gold accents.
